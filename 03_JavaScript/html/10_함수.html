함수(function)

        목표하는 결과를 만들기 위한 코드의 묶음
        코드를 {}(중괄호, 블록)으로 묶어 작성하고 붙여진 이름을 호출해서 사용
        함수 선언, 함수 정의, 함수 호출이라고 부름

        [기본 함수 형태]
        function 함수명() {//함수의 이름 선언
             //함수 정의
        }

        함수명(; //함수를 호출)

        함수란 특정적인 기능을 동작하고자 할 때 특정 기능을 묶어서 만드는 명칭

        예) function 전자레인지 작동시작("몇 초 인가요?") {
            1. 전자레인지 코드를 꼽는다. 
            2. 전자레인지 ("몇")초를 선택한다.
            3. 전자레인지를 ("몇")초만큼 실행한다.
            4. 전자레인지 ("몇")초가 다 되면 종료한다.
        }

매개 변수(parameter)

        *매개 변수(媒介變數, parameter)
         함수 선언시 () 안에 작성되는 변수
          -> 전달 인자로부터 받은 값을 저장
          -> 함수 정의 부분에서 사용 가능 
          媒  중매 매
          介  개입할 개
          變  변할 변
          數  셀 수

          Parameter :변경이 될 기준을 제공
          para- : 근처, 옆에 제공하다
          meter: 기준
          중간 역할을 하거나 전달하는 변수

전달 인자(argument)

    *전달 인자(傳達 引子, argument)
          傳 전할 전
          達  도달될 달
          引 끌어당길 인
          子 아들 자

          Argument : 제공되는 정보, 실제 값
          함수(function)가 호출 될 때 아규먼트의 값들이 증거처럼 파라미터에 전달
            * 요소(element) = <태그>요소값</태그>

무한한 결과가 아닌, 선택지가 있는 결과를 작성할 때 매개변수와 전달인자를 사용 매개변수와 전달인자는 한 세트 매개변수는 전달인자를 전달해주는 전달 역할 일 뿐 전달인자는 위에서 전달인자가 들어옴, 고정적X 매개변수 전달인자 function 결정사 (let 소개팅대상 = "홍길동";) { }

매개변수와 전달인자

return (반환)

        현재 수행중인 함수를 종료하고 호출한 곳으로 돌아감
        호출한 곳으로 돌아갈 때 return 옆에 작성된 값을 가지고 돌아갈 수 있다.

        만약에 return 100; 호출한 곳으로 100을 가지고 돌아갈 수 있음

익명 함수(이름이 없는 함수)

      [기본 함수]
      function 함수명([매개변수]) {
        //함수정의
      }

      함수명(); //이름이 있으니까 함수에서 작성한 함수명으로 언데든지 어디서든 호출 가능

      [익명 함수]
      function([매개변수]) {
        //함수정의
      }
      //함수 이름이 없어서 호출할 수 없음
      기본 함수와 다르게 변수에 대입, 매개 변수로 바로 전달되어야 하는 곳에서 주로 사용

      ex) const 결과 = function(a,b) {
        return a+b;}

화살표 함수(arrow function)

      익명 함수의 한 종류
      익명 함수를 간단히 표기하는 방법
      익명 함수 : function() {}

      1. 기본 형태 : () => {}

      

      2. 매개변수 1개 : (e) => {} 또는 e => {} //() 생략
      3. 매개변수 n개 : (x,y,z) => {} () 생략할 수 없음
